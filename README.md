## Install

```
$ npm install tea-cll
```

## Usage

```
Use official templates
    $ tea <template-name> <project-name>

Use custom templates
    $ tea <username/repo> <project-name>

Use local templates
    $ tea <~/fs/path/to-custom-template> <project-name>

Use git clone
    $ tea -c

Use offline mode
    $ tea -o

Show current version
    $ tea -v

Show help
    $ tea -h
```
